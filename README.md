## Quick Start

To run this app, clone the repository and install dependencies:

```bash
$ git clone https://github.com/A-Diddy/boilerplate.git
$ cd boilerplate
$ npm install

$ cd ../clientApp
$ npm install
```

Then start the server.

```bash
$ npm start
```

Navigate to [http://localhost:3000](http://localhost:3000).
